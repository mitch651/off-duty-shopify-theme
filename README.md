# Off Duty Shopify Theme

Custom Shopify theme for Off Duty (offduty.us).

## Deploy to Shopify

```bash
shopify theme push --theme 176320872612 --allow-live --nodelete
```

Or use the Shopify GitHub integration for automatic deploys (see Shopify Admin > Online Store > Themes > Add theme > Connect from GitHub).
